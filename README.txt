Installation

1. Install composer (if you don't already have it installed)
2. Install yarn (if you don't already have it installed)
3. Create file ".env.local" with configuration for your database
    example: "DATABASE_URL="mysql://db_user:password@127.0.0.1:3306/db_name?serverVersion=mariadb-10.4.8""
4. Go with "composer install" command
5. Go with "php bin/console d:m:m" command
6. Go with "yarn install" command
7. Go with "yarn build" command
8. Enjoy!
